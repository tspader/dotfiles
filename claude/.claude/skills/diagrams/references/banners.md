# Banners and Logos
Large text using block characters and ASCII art.

## Block Character Style
Uses `█` (full block) with `╔ ╗ ╚ ╝ ═ ║` for shading/edges:

```
██╗     ██╗      █████╗ ███╗   ███╗ █████╗
██║     ██║     ██╔══██╗████╗ ████║██╔══██╗
██║     ██║     ███████║██╔████╔██║███████║
██║     ██║     ██╔══██║██║╚██╔╝██║██╔══██║
███████╗███████╗██║  ██║██║ ╚═╝ ██║██║  ██║
╚══════╝╚══════╝╚═╝  ╚═╝╚═╝     ╚═╝╚═╝  ╚═╝
```

Pattern breakdown:
- `█` - solid fill
- `╔` - top-left corner shadow
- `╗` - top-right corner shadow
- `╚` - bottom-left corner shadow
- `╝` - bottom-right corner shadow
- `═` - horizontal shadow edge
- `║` - vertical shadow edge

Construction uses: `|`, `_`, `/`, `\`, backtick, `'`

# Flowcharts

Process flows, decision trees, and state diagrams.

## Basic Flow

```
┌─────────┐     ┌─────────┐     ┌─────────┐
│  Start  │────▶│ Process │────▶│   End   │
└─────────┘     └─────────┘     └─────────┘
```

## Vertical Flow

```
    ┌─────────┐
    │  Start  │
    └────┬────┘
         │
         ▼
    ┌─────────┐
    │ Step 1  │
    └────┬────┘
         │
         ▼
    ┌─────────┐
    │ Step 2  │
    └────┬────┘
         │
         ▼
    ┌─────────┐
    │   End   │
    └─────────┘
```

## Decision Diamond

```
         ┌─────────┐
         │  Start  │
         └────┬────┘
              │
              ▼
        ┌───────────┐
        │ condition │
        └─────┬─────┘
        yes   │   no
    ┌─────────┴─────────┐
    ▼                   ▼
┌───────┐           ┌───────┐
│ True  │           │ False │
└───────┘           └───────┘
```

Compact decision:

```
          ┌───┐
    yes ◀─┤ ? ├─▶ no
          └───┘
```

## Loop

```
    ┌─────────┐
    │  Start  │
    └────┬────┘
         │
         ▼
    ┌────────────┐
┌──▶│  Process   │
│   └─────┬──────┘
│         │
│         ▼
│    ◇─────────◇
│   ╱           ╲
│  ╱   done?     ╲──no──┐
│  ╲             ╱      │
│   ╲           ╱       │
│    ◇────┬────◇        │
│    yes  │             │
│         ▼             │
│    ┌─────────┐        │
│    │   End   │        │
│    └─────────┘        │
│                       │
└───────────────────────┘
```

## Parallel Processes

```
              ┌─────────┐
              │  Start  │
              └────┬────┘
                   │
         ┌─────────┴─────────┐
         ▼                   ▼
    ┌─────────┐         ┌─────────┐
    │ Task A  │         │ Task B  │
    └────┬────┘         └────┬────┘
         │                   │
         └─────────┬─────────┘
                   ▼
              ┌─────────┐
              │  Join   │
              └─────────┘
```

## State Machine

```
            start
              │
              ▼
         ┌─────────┐
    ┌───▶│  IDLE   │◀──────────┐
    │    └────┬────┘           │
    │         │ event          │ done
    │         ▼                │
    │    ┌─────────┐      ┌────┴────┐
    │    │ RUNNING │─────▶│ CLEANUP │
    │    └────┬────┘      └─────────┘
    │         │ error
    │         ▼
    │    ┌─────────┐
    └────│  ERROR  │
         └─────────┘
```

# Character Reference

## Box Drawing (Single Line)

```
Character  Code    Name
─          U+2500  horizontal
│          U+2502  vertical
┌          U+250C  top-left corner
┐          U+2510  top-right corner
└          U+2514  bottom-left corner
┘          U+2518  bottom-right corner
├          U+251C  left tee
┤          U+2524  right tee
┬          U+252C  top tee
┴          U+2534  bottom tee
┼          U+253C  cross
```

Grid:
```
┌ ┬ ┐
├ ┼ ┤
└ ┴ ┘
```

## ASCII Fallback

When Unicode unavailable:
```
Unicode    ASCII
─          -
│          |
┌ ┐ └ ┘    +
├ ┤ ┬ ┴ ┼  +
→          ->
←          <-
█          #
```
